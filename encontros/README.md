# Encontros

Esta pasta organiza as anotações por encontro do Clube Leitura Dev.

## Convenção sugerida

Use a data ISO no nome do arquivo:

- `2026-07-08.md`
- `2026-07-15.md`

## O que registrar

- capítulos ou trechos discutidos;
- perguntas levadas para a conversa;
- principais aprendizados;
- discordâncias, nuances e referências extras;
- próximos passos de estudo.

Use o modelo em `encontros/templates/encontro.md`.
