# Referências

Esta pasta é para materiais complementares ao livro e aos encontros.

- diagramas;
- exemplos de sistemas;
- resumos comparativos com outros livros.

- `referencias/arquitetura-evolutiva/`
- `referencias/estilos-arquiteturais/`
